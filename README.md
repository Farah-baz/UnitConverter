#  UnitConverter

A simple yet powerful Java program to convert between different units — **starting with distance**, and expanding to temperature and weight.  
This tool demonstrates modular coding in Java and effective use of Git branching, issues, and version control.

 Version | Description                                 
---------|---------------------------------------------
 1.0     | Initial distance conversion (km ↔ mi)       
 1.1     | Added temperature conversion (°C ↔ °F)       
 1.3     | Added weight conversion (kg ↔ lb) as an enhancement (Issue #2) 


---

##  Features

-  **Distance Conversion** (Kilometers ↔ Miles) ← _original core feature_  
-  **Temperature Conversion** (Celsius ↔ Fahrenheit)  
-  **Weight Conversion** (Kilograms ↔ Pounds)   
-  Easy to extend with more unit types (e.g., time, volume...)
